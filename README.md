## Preview

Archer allows you to focus more on the game page. It can directly display recommended information at a higher level in the game without needing to pay attention to any other information. Moreover, its installation and use do not require any complicated procedures. You only need to click on the "ğŸš€" icon to start, without any cumbersome additional operations.
