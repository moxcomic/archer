Installing Plugins

  • Place the .arc file into the plugins folder located in the Archer extracted directory.
    • If the plugins folder doesn't exist, you can create one manually.
  • Restart the Archer software, and you will see the newly installed plugin in the Plugin Settings.
  • In the Plugin Settings, toggle the switch button at the bottom right to activate the plugin.
  • There are two icons at the bottom right of the plugin: green and red, which represent the plugin's safe and unsafe statuses, respectively. Plugins marked as safe can be used normally, while plugins marked as unsafe may result in account suspension. Please use them with caution.
