Basic Setup

  • Start Archer and navigate to the Default Settings page in the WebUI.
  • Find the Test Review button and click it.
  • Wait for the Chrome browser to pop up. Open a new Chrome browser tab and visit https://dashboard.hcaptcha.com/signup?type=accessibility&r=524ad0a33287.
  • Enter an email address where you can receive emails.
  • Check your email for the plugin verification email.
  • Right-click to copy the link.
  • Go back to the pop-up window and open the link.
  • Click Set Cookie.
  • When the following message appears, setup is complete.
  • The Cookie is valid for 12 hours.
  • You will need to repeat the process every 12 hours.
